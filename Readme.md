# Kotoba

**Kotoba** is a small cli program that helps to keep record of new words/phrases you come across.

## Installation

	$ npm -g install kotoba

## Usage

	$ kotoba word or phrase

	  recorded: word or phrase
	  today: 21, total: 47


	$ kotoba

	  today: 20, total: 46
	  try 'kotoba --help' for more information
